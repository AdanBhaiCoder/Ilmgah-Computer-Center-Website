Computer Networks

This course provides an introduction to computer networks, with a special focus on the Internet architecture. Topics include layered network architectures, addressing, forwarding, routing, communication reliability, the client-server model, web and email. Besides the theoretical, students acquire practical experience by programming reduced versions of real Internet protocols.
